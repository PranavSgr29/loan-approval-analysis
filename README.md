# 📊 Loan Approval Analytics Dashboard

## 📌 Project Overview

This project is an end-to-end Business Intelligence solution developed using **Python, SQL Server, Power BI, and DAX**. The objective is to analyze **52,000 loan applications** and provide business insights through interactive dashboards.

The project demonstrates the complete data analytics workflow, including data cleaning, SQL analysis, KPI creation, DAX measures, and dashboard development.

## 🛠️ Tools & Technologies

- SQL Server
- SQL
- Power BI
- DAX
- CSV Dataset
